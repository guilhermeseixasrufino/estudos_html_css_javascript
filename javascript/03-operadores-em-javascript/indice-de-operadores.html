Bem-vindo ao Índice de Operadores do Javascrip!

Introdução

Esse pequeno projeto foi criado com o intuito de documentar/organizar meu estudo de operadores em Javascritp, aproveitando para colocar todos os meu conteúdos estudados até o momento em prática.

Aritméticos

+
Adição

Além de concatenação, é utilizado para soma de valores.

exemplo:
5 + 5 = 10
-
Subtração

É utilizado para a subtração de valores.

exemplo:
10 - 5 = 5
*
Multiplicação

É utilizado para a multiplicação de valores.

exemplo:
2 * 5 = 10
/
Divisão

É utilizado para a divisão de valores.

exemplo:
2 / 10 = 5
%
Resto da Divisão

É utilizado para a retornar o resto da divisão inteira.

exemplo:
2 % 10 = 2
**
Potência

É utilizado para realizar a potenciação.

exemplo:
2 ** 4 = 8

Atribuição

+=
Auto Atribuição

Acrescenta o valor somando ao da variável já existente.

exemplo:
n += 4
++
Soma 1

Acrescenta 1 ao valor da variável.

exemplo:
n++
--
Subtraí 1

Subtraí 1 do valor da variável.

exemplo:
n--

Relacionais

>
Maior

Compara o valor, dizendo se é maior

exemplo:
5 > 4 = true
<
Menor

Compara o valor, dizendo se é menor

exemplo:
"5 < 4 = false"
>=
Maior ou Igual

Compara o valor, dizendo se é maior ou igual.

exemplo:
5 >= 5 = true
<=
Menor ou Igual

Compara o valor, dizendo se é menor ou igual.

exemplo:
4 <= 5 = true
==
Igualdade

Compara se os valores são iguais (sem verificar o tipo).

exemplo:
5 == "5" = true
!=
Diferença

Compara se os valores são diferentes (sem verificar o tipo).

exemplo:
5 != "6" = true

Lógicos

!
Negação

Inverte o valor lógico de uma expressão.

exemplo:
!true = false
&&
E (AND)

Retorna true apenas se todas as condições forem verdadeiras.

exemplo:
true && false = false
||
Ou (OR)

Retorna true se pelo menos uma das condições for verdadeira.

exemplo:
true || false = true

Ternário

?:
Ternário

Forma reduzida de um if/else: condição seguida de valor caso verdadeiro e valor caso falso.

exemplo:
idade >= 18 ? "maior" : "menor"
